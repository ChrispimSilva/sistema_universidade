# sistema_universidade
Projeto Integrador 3 do curso ADS do Senac

## Descrição do Projeto
Este projeto faz parte do Projeto Integrador de Analise e Desenvolvimento de Sistemas do Senac e tem por objetivo modelar e prototipar um Sistema Orientado a Objetos, otimizando e organizando a Gestão de dados de uma grande Universidade.

## Objetivos do Sistema
Garantir a organização e rastreabilidade dos cadastros  
Facilitar a gestão acadêmica e administrativa  
Diminuir processos manuais e retrabalhos   

## Protótipo Figma
Foi criado um protótipo navegável apresentando todas as funcionalidades principais, garantindo um alinhamento entre a modelagem teórica e a interface.  
Protótipo foi desenvolvido no Figma, para acessá-lo utilize o link abaixo:

## Funcionalidades Prototipadas
Cadastro de Pessoas Físicas  
Cadastro de Pessoas Juridicas  
Cadastro de Professores  
Cadastro de Alunos  
Cadastro de Fornecedores  
Cada tela apresenta:  
Os campos obrigatórios  
Mensagens de erro  
Navegação alinhada com os fluxos de caso de uso  
Identidade visual  

## Tecnologias Utilizadas
UML  
Figma   
Análise e Modelagem de sistemas  
Metodologia de orientação a objetos
